# homebrew-tap

Homebrew formulae for richard-fairthorne's projects.

## Installation

```bash
brew tap richard-fairthorne/tap
brew install pairqr
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| [pairqr](https://github.com/richard-fairthorne/pairqr) | Pair Android devices for wireless ADB debugging via QR code |
